# Credit Risk Analytics Project

## Business Problem
Banks face losses due to loan defaults. This project analyzes customer data
to identify high-risk segments and predict loan default probability.

## Tools Used
- MySQL
- Python (Pandas, Seaborn)
- Statistics
- Power BI

## Key Analysis
- Overall default rate
- Risk by employment type
- Income and credit score segmentation

## SQL Highlights
- GROUP BY & CASE statements
- Risk segmentation
- Business KPIs

## Outcome
Identified high-risk customer segments which can help banks
reduce default rates by improving approval strategies.
