# IPL Analytics — Match & Player Performance Insights 

## Business Problem
IPL teams, analysts, and fans want to understand:
- Winning strategies
- Player performance
- Toss & venue impact
- Pressure performance in playoffs and finals


##  Tools & Technologies Used
- **SQL** — Match, team & player analysis  
- **Python (Pandas, Matplotlib, Seaborn)** — EDA & visualization  
- **Power BI** — Interactive IPL dashboard  
- **GitHub** — Portfolio hosting  


##  Key Metrics Analyzed
- Matches per season
- Toss winner vs match winner
- Team win counts
- Runs scored & economy rate
- Venue & match stage performance

## Key Insights

### Toss Impact
- Toss winners have a **moderate advantage**
- Toss impact increases in **playoffs and finals**

### Team Performance
- Teams like **MI and CSK** perform consistently well
- Certain teams perform better under **high-pressure matches**

### Batting Performance
- Identified **top run scorers** and power hitters
- Six-hitting ability strongly impacts match outcomes

### Bowling Performance
- Economy rate used to identify **most economical bowlers**
- Bowlers with consistent run control are crucial

### Match Stage Analysis
- Compared **League vs Qualifier vs Final**

## Best Playing XI Logic
Players selected based on:
- Highest total runs (batsmen)
- Lowest economy rate (bowlers)
- Consistency across matches
